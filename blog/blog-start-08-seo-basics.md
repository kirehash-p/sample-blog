## SEOとは何か？

**SEO**（Search Engine Optimization）= 検索エンジン最適化

Googleなどの検索結果で**上位に表示される**ための施策です。

> 「検索結果の1ページ目に入らなければ、存在しないのと同じ」

実際、**90%以上のクリック**は検索結果の1ページ目で発生します。[^1]

[^1]: Backlinko社の調査（2023年）による

## なぜSEOが重要なのか

### トラフィックの内訳

一般的なブログのアクセス源を見てみましょう：

| 流入元 | 割合 | 特徴 |
| --- | --- | --- |
| 検索エンジン | 50〜70% | 継続的・安定的 |
| SNS | 15〜30% | 一時的・バズ依存 |
| ダイレクト | 10〜20% | リピーター |
| その他 | 5〜10% | 被リンクなど |

**検索流入**は最も安定しており、長期的な成長の鍵です。

### 複利効果

良い記事は検索エンジンからの流入を**継続的に**得られます。

$$
\text{累積PV} = \sum_{i=1}^{n} \text{日次PV}_i \approx \text{日次PV} \times n
$$

1日100PVの記事が100本あれば、1日10,000PVです。

## SEOの基本原則

### Google が重視すること

Googleのアルゴリズムは複雑ですが、基本的な原則はシンプルです：

1. **E-E-A-T**: 経験・専門性・権威性・信頼性
2. **ユーザー体験**: 読みやすく、役に立つコンテンツ
3. **技術的な品質**: 高速で、モバイル対応

> 💡 **結論**: 読者のために良いコンテンツを作れば、SEOは後からついてくる

### やってはいけないこと

- ❌ キーワードの詰め込み
- ❌ 他サイトのコピー
- ❌ 隠しテキスト・リンク
- ❌ 質の低い大量生産

これらは**ペナルティ**の対象になります。

## キーワード戦略

### キーワードリサーチ

読者が**実際に検索している言葉**を調べます。

**無料ツール**:
- Google キーワードプランナー
- Ubersuggest
- ラッコキーワード

**有料ツール**:
- Ahrefs
- SEMrush

### キーワードの種類

| 種類 | 例 | 検索数 | 競合 | 成約率 |
| --- | --- | --- | --- | --- |
| ビッグ | Python | 多い | 高い | 低い |
| ミドル | Python 入門 | 中程度 | 中程度 | 中程度 |
| ロングテール | Python 入門 Mac 環境構築 | 少ない | 低い | 高い |

**初心者はロングテールから攻める**のが効果的です。

### 検索意図を理解する

同じキーワードでも、検索意図は異なります：

```
「Python」で検索する人の意図：
├── 🐍 ヘビの「パイソン」について知りたい
├── 💻 プログラミング言語について知りたい
├── 📚 Python を学びたい
└── 🔧 Python 環境を構築したい
```

**検索意図に合った内容**を提供することが重要です。

## 記事のSEO対策

### タイトルタグ

最も重要なSEO要素です。[タイトル術の記事](/sample-blog/blog/blog-start-02-title-techniques.html#クリックされるタイトルの科学)を参照してください。

```html
<!-- 良い例 -->
<title>Python入門 - 初心者向け環境構築ガイド【2025年版】</title>

<!-- 悪い例 -->
<title>私のブログ</title>
```

### メタディスクリプション

検索結果に表示される説明文です。

```html
<meta name="description" content="Python初心者向けに、Mac/Windows/Linuxでの環境構築手順を解説。インストールから最初のプログラム実行まで、画像付きで丁寧に説明します。">
```

**120〜160文字**を目安に、記事の内容を要約します。

### 見出し構造

見出しは**階層的に**使います。

```markdown
# h1: 記事タイトル（1つだけ）

## h2: セクション1
### h3: サブセクション
### h3: サブセクション

## h2: セクション2
### h3: サブセクション
```

### 内部リンク

関連記事へのリンクは重要なSEOシグナルです。

```markdown
詳しくは [Markdownの書式テクニック](/sample-blog/blog/blog-start-04-formatting.html) を参照してください。
```

### 画像の最適化

[画像の活用法の最適化の節](/sample-blog/blog/blog-start-05-images.html#画像の最適化)で学んだように：

- **alt属性**を適切に設定
- **ファイルサイズ**を最適化
- **ファイル名**にキーワードを含める

```html
<!-- 良い例 -->
<img src="python-setup-guide.png" alt="PythonのインストールとVSCode設定画面">

<!-- 悪い例 -->
<img src="IMG_20250101.png" alt="">
```

### モバイル対応

2019年以降、Googleは**モバイルファーストインデックス**を採用しています。

$$
\text{検索順位} \propto \text{モバイルでの使いやすさ}
$$

- レスポンシブデザイン
- タップしやすいボタンサイズ
- 読みやすいフォントサイズ

### 構造化データ

検索結果にリッチスニペットを表示できます。

```json
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "Python入門ガイド",
  "author": {
    "@type": "Person",
    "name": "著者名"
  },
  "datePublished": "2025-10-21",
  "image": "https://example.com/image.jpg"
}
```

## コンテンツ戦略

### 柱コンテンツ（Pillar Content）

メインとなる長文記事を「柱」として、関連記事からリンクします。

```
        ┌─────────────────┐
        │  柱コンテンツ    │
        │ (総合ガイド)     │
        └────────┬────────┘
                 │
     ┌───────────┼───────────┐
     ▼           ▼           ▼
┌─────────┐ ┌─────────┐ ┌─────────┐
│ 詳細記事1│ │ 詳細記事2│ │ 詳細記事3│
└─────────┘ └─────────┘ └─────────┘
```

### 定期的な更新

古い情報は検索順位が下がります。

```markdown
---
title: "Python入門ガイド【2025年版】"
date: "2024-01-15"
updated: "2025-10-21"  # ← 定期的に更新
---
```

## SEOの効果測定

### Google Search Console

無料で使える必須ツールです。

確認できること：
- 検索クエリ（どんな言葉で検索されているか）
- クリック数・表示回数
- 平均掲載順位
- インデックス状況

### 重要な指標

| 指標 | 説明 | 目標 |
| --- | --- | --- |
| インプレッション | 検索結果に表示された回数 | 増加傾向 |
| CTR | クリック率 | 2〜5% |
| 平均掲載順位 | 検索結果での順位 | 10位以内 |

## よくある失敗

### ❌ 失敗1: キーワード偏重

```markdown
<!-- 悪い例：不自然なキーワード詰め込み -->
Python入門でPythonを学ぶ。Python初心者がPythonを始めるなら
Python入門サイトでPythonを勉強しましょう。
```

### ❌ 失敗2: 短すぎる記事

300文字程度の記事では、検索上位は難しいです。

**目安**: 1500〜3000文字以上

### ❌ 失敗3: SEOだけを考える

> **読者のため**のコンテンツが、結果的にSEOにも良い

読者を無視したSEO対策は、長期的には逆効果です。

- [x] 1500文字以上あるか
- [x] 読者の疑問に答えているか

## まとめ

SEOの基本ポイント：

| ポイント | 説明 |
| --- | --- |
| コンテンツ第一 | 読者のための良い記事を書く |
| キーワード | ロングテールから攻める |
| 技術面 | 速度・モバイル対応を確保 |
| 継続 | 定期的に更新・改善する |

SEOは**マラソン**です。地道に続けることで、確実に成果が出ます 🏃
